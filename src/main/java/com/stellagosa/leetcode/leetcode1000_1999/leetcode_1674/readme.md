对于任意一堆 nums[i] 和 nums[n-i-1] </br>
假设目标值是 k，arr[k] 表示目标值是 k 时，nums 互补的最小操作数 </br>
令 a=nums[i], b=nums[n-i-1], 假设 a<b </br>
1<=nums[i]<=limit </br>
k∈[2,2*limit], a∈[1,limit], b∈[1,limit]

* 2 <= k < a+1，即使b变成1，数对和是a+1，依然大于k，还需要修改a的值，所以 arr[k]+2
* a+1<= k < a+b，只需修改b的值就可以了，所以 arr[k]+1
* k=a+b，不需要修改值，所以 arr[k]+0
* a+b < k <= b+limit,只需修改a的值即可，所以 arr[k]+1
* b+limit < k <= 2*limit，即使将a的值改成limit，仍然需要修改b的值，所以arr[k]+2

每次修改arr，都需要修改下标在一个范围内的值，可以构建差分数组，每次只需修改两个值

构建数组arr的差分数组diff[i]=arr[i]-arr[i-1]



# 差分数组
数组 a1,a2,a3,...,an  </br>
构建新数组b，令 b1=a1,b2=a2-a1,b3=a3-a2,...,bn=an-a_n-1
数组b就是数组a的差分数组，根据数组b可以轻松还原出原数组a </br>
若对原数组a的 [i,j] 位置修改，例如全部减c，则在差分数组b中，只需将bi-c,b_j+1-c，

示例
1，3，5，7，9，11
1，2，2，2，2，2

将下标2到4的值减1
1，3，4，6，8，11
1，2，1，2，2，3
在差分数组中，只需将下标2的值减1，下标5的值加1
