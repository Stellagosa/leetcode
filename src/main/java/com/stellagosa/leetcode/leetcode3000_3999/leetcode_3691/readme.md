# 3691. 最大子数组总值 Ⅱ

给定一个长度为 n 的整数数组 nums 和一个整数 k。

你必须从 nums 中选择 恰好 k 个非空子数组 nums[l..r]。子数组可以重叠，但同一个子数组（相同的 l 和 r）不能 被选择超过一次。

子数组 nums[l..r] 的 值 定义为：max(nums[l..r]) - min(nums[l..r])。

总值 是所有被选子数组的 值 之和。

返回你能实现的 最大 可能总值。

子数组 是数组中连续的 非空 元素序列。

### 示例 1:
> 输入: nums = [1,3,2], k = 2
>
> 输出: 4
>
> 解释:
>
> 一种最优的方法是：
>
>- 选择 nums[0..1] = [1, 3]。最大值为 3，最小值为 1，得到的值为 3 - 1 = 2。
>- 选择 nums[0..2] = [1, 3, 2]。最大值仍为 3，最小值仍为 1，所以值也是 3 - 1 = 2。
>
> 将它们相加得到 2 + 2 = 4。

### 示例 2:
> 输入: nums = [4,2,5,1], k = 3
>
> 输出: 12
>
> 解释:
>
> 一种最优的方法是：
>
>- 选择 nums[0..3] = [4, 2, 5, 1]。最大值为 5，最小值为 1，得到的值为 5 - 1 = 4。
>- 选择 nums[1..3] = [2, 5, 1]。最大值为 5，最小值为 1，所以值也是 4。
>- 选择 nums[2..3] = [5, 1]。最大值为 5，最小值为 1，所以值同样是 4。
>
> 将它们相加得到 4 + 4 + 4 = 12。

### 提示:
- 1 <= n == nums.length <= 5 * $10^4$
- 0 <= nums[i] <= $10^9$
- 1 <= k <= min($10^5$, n * (n + 1) / 2)

### 思路

#### ST 表 

1. 给定数组 nums， O(1) 时间复杂度求 [a,b] 范围内的最大值。

设 $ f(i,j) = [i, i+2^j-1]_{max}$

则
$$
\begin{align}
f(i,j-1) &= [i, i+2^{j-1}-1]_{max}
\end{align}
$$

则
$$
\begin{align}
f(i,j) &= [i, i+2^j-1]_{max} \\
&= [i, i+2^{j-1}+2^{j-1}-1]_{max} \\
&= \{[i, i+2^{j-1}-1]_{max},[i+2^{j-1},(i+2^{j-1})+2^{j-1}-1]_{max}\}_{max} \\
&= [f(i,j-1),f(i+2^{j-1},j-1)]_{max}
\end{align}
$$

2. 举例

给定数组 3，5，8，1，7，12，2，4，求任意范围内的最大值

$f(i,0) = [i,i]_{max}=a_i$

$f(i,1) = [f(i,0),f(i+1,0)]_{max}$

$f(i,2) = [f(i,1),f(i+2,1)]_{max}$

$f(i,3) = [f(i,2),f(i+4,2)]_{max}$

填表如下：

| i/j | 0  | 1  | 2  | 3  |
|-----|----|----|----|----|
| 0   | 3  | 5  | 8  | 12 |
| 1   | 5  | 8  | 8  |    |
| 2   | 8  | 8  | 12 |    |
| 3   | 1  | 7  | 12 |    |
| 4   | 7  | 12 | 12 |    |
| 5   | 12 | 12 |    |    |
| 6   | 2  | 4  |    |    |
| 7   | 4  |    |    |    |


现在求 $[2,6]_{max}$

这个子数组的长度是  6-2+1=5

现令 $2^k <= 5$，求 k 的最大值

$5 = (101)_2$  二进制长度是3， k = 3 - 1 = 2

现在 [2,6] 可以分成长度为 $2^k$ 两部分计算，分别是 
- $[2,2 + 2^k - 1] = [2,5]$
- $[6-2^k+1,6] = [3,6]$

最后求这两部分的最大值就可以了

$$
\begin{align}
[2,6]_{max} &=\{[2,2+2^2-1]_{max},[6-2^2+1,6]_{max}\}_{max} \\
&=[f(2,2),f(3,2)]_{max} \\
&=[12,12]_{max} \\
&=12 \\
\end{align}
$$

同理，最小值也可以使用 ST 表

#### 本题思路

对于任意子数组 $[i,j](i<=j)$，当固定 i 的值不变，随着 j 值增大，子数组 $[i,j]$ 的值是非递减的。

假设数组长度为 n，那么我们可以将子数组分成 n 组

- [0,n-1],[0,n-2],[0,n-3]...[0,3],[0,2],[0,1],[0,0]
- [1,n-1],[1,n-2],[1,n-3]...[1,3],[1,2],[1,1]
- [2,n-1],[2,n-2],[2,n-3]...[2,3],[2,2]
- [3,n-1],[3,n-2],[3,n-3]...[3,3]
- ...
- [n-3,n-1],[n-2,n-2]
- [n-1,n-1]

这 n 组子数组，每一组都是非递增的，所以原问题也就变成了 n 组非递增数列合并取最大的 k 个。

我们可以维护一个大顶堆，将每一组第一个最大的值加入堆中，然后弹出最大的值，加入结果中，再将改组第二大的值加入堆中，以此往复，直到弹出需要的 k 个值结束。
